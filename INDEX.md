## 🔧 File Maintenance Guide

### What to Update When You Modify Code

#### If you add a new response pattern:
1. ✅ Update chatbot.py (add elif clause)
2. ✅ Update README.md (features list)
3. ✅ Update CODE_EXPLANATION.md (explain new logic)
4. ✅ Update SAMPLE_OUTPUT.md (add example)
5. ⚠️ Check ENHANCEMENTS.md (remove if implemented from there)

#### If you add a new feature:
1. ✅ Update chatbot.py
2. ✅ Update README.md (features + usage)
3. ✅ Update CODE_EXPLANATION.md (detailed walkthrough)
4. ✅ Update SAMPLE_OUTPUT.md (demonstrate feature)
5. ✅ Update PROJECT_SUMMARY.md (architecture changes)
6. ✅ Update QUICKSTART.md (if affects running)

#### If you fix a bug:
1. ✅ Update chatbot.py
2. ✅ Update SAMPLE_OUTPUT.md (correct expected output)
3. ⚠️ Check CODE_EXPLANATION.md (if logic changed)
